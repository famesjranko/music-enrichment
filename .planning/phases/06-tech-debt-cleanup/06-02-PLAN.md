**CoverArtArchiveApi migration:**

The CAA API has 3 HTTP calls:
1. `getArtworkUrl` -- uses `httpClient.fetchRedirectUrl(url)`. Leave this as-is since fetchRedirectUrl returns a URL string, not JSON. The redirect pattern is fundamentally different from JSON fetches.
2. `getGroupArtworkUrl` -- same, uses `fetchRedirectUrl`. Leave as-is.
3. `getArtworkMetadata` -- uses `httpClient.fetchJson(url) ?: return null`. Migrate this one:
   ```kotlin

       is HttpResult.Ok -> result.body
       else -> return null
   }
   ```
