<objective>
Add Fanart.tv album-specific art endpoint for faster ALBUM_ART lookups and ListenBrainz similar-artists endpoint as a SIMILAR_ARTISTS fallback at priority 50.

Purpose: Fanart.tv currently fetches the full artist response for album art, which includes all albums for that artist. The album-specific endpoint (`/v3/music/albums/{releaseGroupMbid}`) returns only the target album's covers, which is faster and more targeted. ListenBrainz has a similar-artists endpoint (`/1/explore/lb-radio/artist/{mbid}/similar`) that works without an API key, providing a free fallback behind Last.fm (priority 100) for SIMILAR_ARTISTS.

Output: Fanart.tv album-first strategy with artist fallback. ListenBrainz SIMILAR_ARTISTS at priority 50. Both with tests.
</objective>
